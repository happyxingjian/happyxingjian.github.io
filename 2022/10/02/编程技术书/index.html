技术书

    • 《Java 并发编程之美》
    • 《实战 Java 高并发程序设计》
    • 《Java 并发编程的艺术》

  • 代码优化：
    • 《代码整洁之道》
    • 《重构：改善既有代码》
    • 《Effective java中文第三版》
    • 《阿里巴巴 Java 开发手册》
    • 《代码大全》

  • 操作系统：
